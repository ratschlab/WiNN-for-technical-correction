This notebook reproduces the shareable public-data preprocessing described in Public multi-batch measurement benchmark and Supplementary Methods S7. It expects the public MetaboLights file Dataset07__SFPM.xlsx, applies the manuscript’s missing-value handling procedure, and writes the processed CSV files used for downstream analysis.

Users should normally download the spreadsheet from the MTBLS79 record at MetaboLights and place it at data/public/raw/Dataset07__SFPM.xlsx.

1 1. Resolve the input dataset

dataset_path <- params$dataset_path

if (!file.exists(dataset_path) && isTRUE(params$download_if_missing)) {
  dir.create(dirname(dataset_path), showWarnings = FALSE, recursive = TRUE)
  utils::download.file(
    url = params$direct_download_url,
    destfile = dataset_path,
    mode = "wb",
    method = "libcurl",
    quiet = TRUE
  )
}

if (!file.exists(dataset_path)) {
  stop(
    "Dataset07__SFPM.xlsx was not found at ", dataset_path, ".\n",
    "Download it from ", params$record_url, " and rerun the notebook."
  )
}

2 2. Run the manuscript preprocessing workflow

source(file.path("..", "scripts", "preprocess_mtbls79_public_data.R"))

public_data <- preprocess_mtbls79_public_data(
  file_path = dataset_path,
  output_dir = file.path("..", "data", "public", "processed"),
  seed = 42L,
  reuse_existing = TRUE,
  verbose = FALSE
)

meta <- public_data$raw_metadata
raw_data <- public_data$raw_data
imputed_data <- public_data$imputed_data
feature_summary <- public_data$feature_summary
