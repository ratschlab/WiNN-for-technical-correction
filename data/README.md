# Data Layout

- `simulated/` contains generated benchmark files created by `scripts/simulate_metabolomics_benchmark.R`.
- `public/raw/` is the expected location for `Dataset07__SFPM.xlsx` downloaded from the public MTBLS79 MetaboLights record.
- `public/processed/` is created by `scripts/preprocess_mtbls79_public_data.R` and stores the imputed CSV outputs.
